# Golden Rose Web Development Project

## Overview
Golden Rose is a web development project designed and implemented as part of a Web Programming course. The project demonstrates the application of recommended web design practices and core web development concepts through a multi-page interactive website.

## Project Goals
- Apply web development principles and best practices
- Design a consistent and user-friendly multi-page website
- Implement interactivity using JavaScript
- Demonstrate teamwork and individual contribution

## Features
- Multi-page website with a unified layout and navigation menu
- Home page with header and footer
- Interactive forms for user input
- Internal and external page linking
- Email link integration
- Image-rich content for enhanced user experience
- Tables and lists for structured content presentation
- Consistent color scheme and styling across all pages

## Technologies Used
- HTML5
- CSS3
- JavaScript

## JavaScript Functionality
- Conditional logic (if / else, switch)
- Loops (for, while)
- User-defined functions
- Arrays and basic data manipulation
- Interactive elements to enhance user experience

- Home page connecting all web pages via navigation menu
- Individual pages developed by each team member
- Unified styling and design across the website
